# API Endpoints

## Auth
| Method | Endpoint | Роль |
|--------|----------|------|
| POST | /api/auth/register | - |
| POST | /api/auth/login | - |
| GET | /api/auth/users | tex, gexp |
| POST | /api/auth/users/update-role | tex |

## Admin (tex only)
| Method | Endpoint | Описание |
|--------|----------|---------|
| POST | /api/admin/chief-experts | Создание главного эксперта |
| POST | /api/admin/participants | Создание участника |
| DELETE | /api/admin/users | Удаление пользователя |
| GET | /api/admin/users | Список всех пользователей |
| POST | /api/admin/events | Создание мероприятия |
| POST | /api/admin/modules | Создание модуля |
| DELETE | /api/admin/events | Удаление мероприятия |
| DELETE | /api/admin/modules | Удаление модуля |
| GET | /api/admin/events | Список всех мероприятий |
| GET | /api/admin/events/:eventId | Мероприятие по ID |

---

## Response Success Format
```json
{
  "success": true,
  "message": "...",
  "user/event/module": {...}
}
```

## Response Error Format
```json
{
  "success": false,
  "message": "..." 
}
```

---

## Roles
- tex: Технический эксперт
- gexp: Главный эксперт
- exp: Эксперт
- part: Участник

---

## Headers
```
Authorization: Bearer <token>
Content-Type: application/json
```

---

## HTTP Status Codes
- 200: OK
- 201: Created
- 400: Bad Request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 500: Server Error
